# Sistema 2GDL con IA

Implementación de un esquema de control basado en inteligencia artificial para una plataforma móvil omnidireccional con mecanismo de dos grados de libertad.

## Descripción
Este proyecto forma parte del trabajo de titulación en la Universidad de las Fuerzas Armadas ESPE, orientado al desarrollo de un sistema robótico líder-seguidor.

## Tecnologías
- ROS 2 Humble
- Python / C++
- Dynamixel SDK
- ZED 2 Camera
